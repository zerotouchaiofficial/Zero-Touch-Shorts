# Security Policy

## Reporting a Vulnerability
If you discover a security issue in this project, please **do not open a public issue**.  
Instead, report it privately via email to: **zerotouchai.official@gmail.com**  

Include the following information:  
- A clear description of the vulnerability  
- Steps to reproduce the issue  
- Any logs or screenshots that help explain the problem  
- Your environment (OS, Python version, etc.)

## Supported Versions
This project is actively maintained. Security fixes are applied to the latest version.  
Make sure you are using the most recent release to avoid known vulnerabilities.

## Security Best Practices for Users
- **Do not commit API keys, secrets, or OAuth tokens** to the repository.  
- Use GitHub Secrets or environment variables for sensitive information.  
- Limit access to accounts used with this tool.  
- Always verify third-party dependencies for security risks.
